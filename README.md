**6. Why Keratin Is NOT Dead Weight**
Every functional group on keratin participates in conduction:

–NH₂   (Lys)         → Proton relay station

–C=O   (peptide)     → Grotthuss hopping site
–S–S–  (disulfide)   → Additional relay path
